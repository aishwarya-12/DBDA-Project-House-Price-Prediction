🏡 Zillow House Price Prediction — End-to-End Scalable ML Pipeline
An end-to-end data science and data engineering project that predicts house prices using Zillow data.
The project demonstrates a complete production-style ML workflow including data collection, preprocessing, synthetic data generation, scalable model training, experiment tracking, interactive applications, analytics dashboards, and cloud deployment.

The goal was not only to build an accurate model but also to design a scalable and reproducible machine learning pipeline using modern data engineering and MLOps practices.

📖 Project Overview
Real estate pricing depends on multiple factors such as location, size, and property characteristics. This project aims to build a machine learning system capable of predicting housing prices based on such features.

To simulate real-world production workflows, the system incorporates:  
        Automated data collection through web scraping API  
        Data preprocessing and feature engineering  
        Synthetic data generation to improve training scalability  
        Distributed machine learning with PySpark  
        Experiment tracking with MLflow  
        Interactive user interface using Streamlit  
        Business intelligence dashboards with Tableau  
        Cloud deployment on AWS EC2  
This project combines Data Science, Data Engineering, and MLOps practices to create a production-style ML system.
